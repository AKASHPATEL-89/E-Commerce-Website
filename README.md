## How to Use

### 1. Setup
1. Download all files to your computer
2. Keep the folder structure intact
3. Open `index.html` in any modern web browser

### 2. Customization

#### Changing Colors
1. Open `css/style.css`
2. Find the CSS custom properties or gradient definitions
3. Modify the color values to match your brand

#### Updating Content
1. Open `index.html`
2. Replace text content with your own information
3. Update contact details, services, and company information

#### Adding Images
1. Place images in the `images/` folder
2. Update the `src` attributes in HTML
3. Replace placeholder content with real images

#### Modifying Services
1. Edit the services section in `index.html`
2. Update service titles, descriptions, and feature lists
3. Change icons by updating Font Awesome classes

### 3. Deployment Options

#### Local Hosting
- Simply open `index.html` in a web browser
- Works offline once loaded

#### Web Hosting
- Upload all files to any web hosting service
- Maintain the folder structure
- Access via your domain/hosting URL

#### GitHub Pages
- Create a GitHub repository
- Upload all files
- Enable GitHub Pages in repository settings

#### Netlify/Vercel
- Drag and drop the entire folder to Netlify or Vercel
- Get instant deployment with custom domain options

## Customization Guide

### Adding New Sections
1. Add HTML structure in `index.html`
2. Add corresponding styles in `css/style.css`
3. Update navigation menu if needed
4. Add any required JavaScript functionality

1. Open `css/style.css`
2. Find transition and animation properties
3. Modify duration values (e.g., `0.3s` to `0.5s`)

### Adding New Portfolio Items
1. Copy an existing portfolio item in HTML
2. Update the `data-category` attribute
3. Change the content and links
4. Ensure the category filter includes the new category

### Modifying Form Behavior
1. Open `js/script.js`
2. Find the `handleContactSubmit` function
3. Modify validation rules or submission logic
4. Update success/error messages
